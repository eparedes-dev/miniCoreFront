Pasos para ejecutar Angular funcionamiento de front:

1. Instalacion de Node.js en caso de que no se tenga se debe instalar
2. Instalacion del CLI de Angular ejecutando el comando: npm install -g @angular/cli
3. Ejecutar el comando npm update en la raiz del proyecto
4. Para la ejecucion del proyecto ejecutar ng serve

Se debe tener en cuenta que al recibir los datos mediante API se 
este llamando a la URl correcta en los archivos de vendedor.service. ts y venta.service.ts
